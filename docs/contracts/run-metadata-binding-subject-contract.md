# Run Metadata Binding Subject Contract Draft

Status: inactive `run-metadata-binding-subject/0.2.0-draft`

This subject binds one exact `run_id` to one `programme_month` and exact consumer. Programme month must never be parsed from `run_id`.

Its generic envelope uses purpose and subject type `run-metadata-binding` and ordinary anchored snapshot membership. Effective values must equal the registry snapshot, monthly selection, and any future manifest. Only fictional year-2099 examples are authorized; real run metadata remains blocked.
